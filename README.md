# 知乎小报

知乎小报是一款基于Android平台开发的知乎日报第三方客户端。内容来源于知乎日报官方，API来自 [izzyleung](https://github.com/izzyleung)。如果本App存在侵犯知乎权益的嫌疑，那么本人将删除整个项目。请您暸解相关情况，并遵守知乎协议。

App内容主要分为4部分:最新日报、主题日报、热门日报、历史日报。
